# 水印相机

📱 一个功能完整的Android水印相机应用，支持日期、星期、GPS定位和天气水印。

## 主要功能

- ✅ 拍照功能（支持前后摄像头切换）
- ✅ 水印信息：日期、星期、时间、GPS定位、天气
- ✅ 自动保存到相册
- ✅ 简洁美观的界面

## 技术栈

- React Native 0.81.5
- Expo SDK 54
- Expo Camera
- Expo Location
- Expo Media Library

## 构建APK

本仓库已配置GitHub Actions自动构建。

1. 点击 **Actions** 标签
2. 选择 **Auto Build APK**
3. 点击 **Run workflow**
4. 等待完成，下载APK

## 直接安装

下载APK文件到Android手机，点击安装即可。
